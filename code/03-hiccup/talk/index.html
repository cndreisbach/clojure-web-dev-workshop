2.2.2 Libraries    slide

The Clojure way: lots of loosely coupled libraries.

Pros: flexible, able to move fast.

Cons: instead of knowing 4 languages + a framework, now you get to learn a lot of libraries.

  • Ring
  • Compojure
  • libnoir
  • Hiccup
  • Garden
  • ClojureScript
  • Liberator
  • Friend
  • Korma
